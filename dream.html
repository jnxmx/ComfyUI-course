Сон

Выберите (вспомните, выдумайте) один из ваших снов. Не важно, случайный или очень важный для вас. Представьте его себе как хронологическую серию изображений с событиями либо образами.

На основе этих идей вам необходимо создать серию из 3–10 изображений, объединённых общим визуальным решением. Это домашнее задание по созданию статической серии.

  1. 1. Решение по стилю

    Для создания устойчивого авторского стиля используйте комбинацию инструментов: подберите подходящие LoRA, составьте постоянную часть промпта, и задействуйте визуальные адаптеры или референсы. Важно, чтобы найденная стилистика сохранялась на протяжении всей серии.

  2. 2. Решение по форме

    Контроль композиции и объектов должен осуществляться на основе ваших заранее подготовленных набросков (фотографий или рисунков). Используйте их в качестве контрольных изображений для ControlNet или как reference, чтобы точно перенести задуманную форму в генерацию.

  3. 3. Генерация с помощью Krea + Redux & ControlNet

    Основной воркфлоу для создания изображений с глубоким контролем стиля и формы.

    Не забудьте установить ноды для Redux через Manager — Install Missing Custom Nodes.

    Flux.1 Krea Full Workflow → JSON

  4. 4. Klein 9B Distilled (альтернативно)

    Альтернативный инструмент для генерации и редактирования изображений.

    Edit Image Klein 9B Distilled → JSON

  5. 5. Flux.1 Fill (для инпейнтинга)

    Используйте этот воркфлоу для исправления или добавления деталей в уже созданные изображения.

    Flux.1 Fill Inpaint → JSON

  6. 6. Klein 9B (для edit инпейнтинга)

    Инструмент для точечного редактирования через маски (Edit Inpainting).

    Klein 9B Masked Inpaint → JSON

Результат выгружайте в галерею с подписью #сон.

Мы ещё вернёмся к этим изображениям во второй половине курса!
