СОН

Выберите (вспомните, выдумайте) какой-нибудь из ваших снов. Не важно, случайный или очень важный для вас. Представьте его как хронологическую серию изображений с событиями или образами, которые станут основой короткой анимации.

  1. 1. Исходное изображение

    Выберите единые пропорции для каждого кадра и подготовьте несколько (2–3) кадров будущего видео. Чтобы удерживать стиль используйте LoRA и постоянную часть промпта. Пробуйте задействовать ControlNet: наброски из пятен можно превращать с помощью Upscaler ControlNet, а для карт глубины и контуров используйте Union ControlNet.

    Flux Krea Big Workflow → JSON

  2. 2. Промежуточные кадры через Qwen Edit

    С помощью Qwen Edit сгенерируйте дополнительные кадры, которые будете использовать в качестве контрольных для генерации видео от первого кадра к последнему.

    QWEN Edit Plus — Next Scene Lora → JSON

  3. 3. Обогащённые промпты

    Языковой моделью автоматически добавьте деталей в промпты для переходов между кадрами.

    Prompt Detailer Qwen VL → JSON

  4. 4. Генерация клипов

    Сгенерируйте несколько клипов для ваших кадров. Либо переходами от первого кадра к последнему, либо просто I2V.

    Wan 2.2 I2V or FLF2V → JSON

  5. 5. Финальная анимация в VACE

    В VACE сшиваем всё в финальную единую анимацию (при необходимости — зацикленную).

    WAN 2.2 VACE Connect Videos → JSON
